# Security Policy

## Supported Versions

The following versions of the platform receive active security updates:

| Version | Supported |
| :--- | :---: |
| 1.0.x | Yes |
| < 1.0.0 | No |

---

## Reporting a Vulnerability

We take the security of our platform seriously. If you find a security vulnerability, please report it immediately rather than opening a public issue.

### Process
1. Email your findings to **security-alert@company.com**.
2. Include a detailed description of the vulnerability, step-by-step instructions to reproduce it, and any proof-of-concept scripts.
3. We will acknowledge receipt of your report within **24 hours**.
4. We will coordinate a response plan and aim to release a patch or mitigation within **7 days**.
5. Please follow responsible disclosure principles and do not share details publicly until a fix is released.
